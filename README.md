# Training Philosophies

Training philosophy frameworks and methodology documentation for Gravel God Coaching.

## Contents

| File | Description |
|------|-------------|
| `ENDURANCE_TRAINING_MODELS.md` | 13 endurance training methodologies with detailed analysis |
| `PHILOSOPHY_MATRIX.md` | Decision matrix for matching athletes to philosophies |
| `PHILOSOPHY_SELECTION.md` | 5-step heuristic framework for philosophy selection |
| `training_systems_comparison.csv` | Comparative data across training systems |

## Training Methodologies Covered

1. Traditional Pyramidal Periodization
2. Polarized Training (80/20)
3. Sweet Spot/Threshold Training
4. High-Intensity Interval Training (HIIT)
5. Block Periodization
6. Reverse Periodization
7. Autoregulated/HRV-Based Training
8. MAF/Low-HR Training
9. Critical Power/W' Model
10. INSCYD Metabolic Profiling
11. Double-Threshold (Norwegian Model)
12. HVLI/LSD-Centric
13. UETS (Unified Endurance Training System)
